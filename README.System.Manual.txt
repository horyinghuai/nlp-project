============================================================

1. INTRODUCTION
----------------
The Smart Resume Analyzer Portal is a web-based application that uses Natural 
Language Processing (NLP) techniques to extract key information (Name, Email, 
Skills, Education) from resume PDF files. It includes an interactive chatbot 
to assist users.

2. SYSTEM REQUIREMENTS 
-----------------------
- Operating System: Windows, macOS, or Linux
- Python Version: Python 3.8 or higher
- Web Browser: Chrome, Edge, or Firefox

3. INSTALLATION INSTRUCTIONS
----------------------------
Step 1: Unzip the project folder 'Resume_Portal'.
Step 2: Open your terminal/command prompt.
Step 3: Navigate to the project directory:
    cd path/to/Resume_Portal
Step 4: Install the required Python libraries:
    pip install -r requirements.txt

4. HOW TO RUN THE SYSTEM
------------------------
Step 1: In the terminal, run the application:
    python app.py
Step 2: You will see a message "Running on http://127.0.0.1:5000".
Step 3: Open your web browser and go to: http://127.0.0.1:5000

5. USER GUIDE
-------------
- Uploading: Click "Choose File", select a PDF resume, and click "Analyze Resume".
- Results: Extracted information will be displayed on a new page.
- Chatbot: Click the "Chat" bubble in the bottom right corner to ask questions.

6. CHATBOT COMMANDS (FAQs)
--------------------------
Try asking these questions:
- "How do I upload a resume?"
- "What formats are supported?"
- "How does the NLP work?"
- "Who created this system?"
- "Is my data safe?"
